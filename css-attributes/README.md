CSS Attributes
==========================

## Attribute's Order

In CSS the order of attributes should be in alphabetic order.

**Correct Order** :ok_hand:
```
header{
    background-color: #2a437a;
    margin: auto;
    margin-top: 50px;
    margin-left: 50px;
    margin-right: 50px;
    width: 800px  
}
```

**Incorrect Order** :-1:
```
header{
    margin: auto;
    width: 800px  
    background-color: #2a437a;
    margin-top: 50px;
    margin-left: 50px;
    margin-right: 50px;
}
```
